Para jugar, instalar la libreria de python 3 PyGame y correr con python 3 el archivo game.py, estando ubicado en la misma carpeta que el resto de archivos
Por problemas técnicos, el juego solo se ha probado en windows, por lo que se recomienda utilizarlo en el mismo, pero debería poder funcionar en linux sin problemas
*********************
Instrucciones: Girar nave con las flechas derecha e izquierda, avanzar y retroceder con las flechas de arriba y abajo respectivamente, no se puede girar mientras se avanza/retrocede. Se dispara con el espacio. 
Para salir del juego, apretar escape.
